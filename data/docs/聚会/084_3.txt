 { 塔羅兩大功能 }
一. 和心靈直接溝通(治療)了解、分析和解除心理矛盾阻障，重得快樂。
二. 占卜對事情分析(預測)預知未來、分析問題、建議選擇。
 { 預測內容 }
一. 你可選擇三大主題，事業、感情、健康，再選擇問題方式。
二. 占卜多建議問三個月內的事情，更遠的可選用命盤問命。
三. 選用「單牌問事」或「牌陣分析」作合適服務。
