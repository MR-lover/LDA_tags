{ 用途 } 
來報讀的同學們一般有三種需求： 
"我的工作環境和社交上遇到很多的人和事，但我真不知道誰是好人和壞人。" 
面相學的普遍應用，是在人與人的環境裡，辨善惡，辨智愚。無論在人事管理上或在交朋結友上，都能讓你果斷地作出準確的決定。什麼樣子的人會偽善？什麼樣子的人最誠實而可靠？什麼樣子的人配上什麼性格和能力？本課程會詳細為你解讀。 
"我的人生經歷起伏很大，我想了解命運是否存在。" 
面相學能解釋包括性格和命運。什麼時候運順？什麼時候運阻？面相學是易學大門的其中一個分科，和八字算命相似，面相學有抽象性的命理解讀。我們正在尋找人生的規律。 
"我很喜歡學習，我更喜歡心理學、宗教、哲學和神祕的主題。" 
在美國大學時期，老師也參加很多各門宗教的聚會，希望了解他們存在的意義，其中覺得佛學思維最值得研究。家裡也收藏了很多命理的書。我們課裡可以多交流！ 
{ 教材特色 } 
教材的發展源於多方的努力，前輩的命理學者，新聞攝影師，牆外內主流媒體，牆外高清圖庫，資料分析員，繪圖員，幻燈片排版的圖像設計師，缺一不可。文字方面有講師的新筆經驗總結和其他老師的文筆分享(必注名出處)。圖像式教學是相學教育界中的緊缺的，打破了先師以口傳的傳統保守態度，以學理和驗證為學習方式。

{ 浪仁的經歷 } 
香港和米国兩地长大，14歲移居米国加州生活十年，学仕毕业於SanJose State Univerty主修Studio Art(photography and Graphics)，在大学时期开始对宗教哲学和各地文化感兴趣，參加各宗教活動(Hinduism, Islam, Christian, Judaism, Buddhism)，初步學佛。高中時一次偶然，接受了朋友親戚為我作占卜，預測結果成為了事實。高中畢業年開始自修塔羅，於大學内开始擺攤塔罗占卜，同時參與社區自發性的靈修類活動。開始了素食生活。 
{ 緣起 } 
2012年10月開始教材開發，設計教程、參考書藉、編寫講義、手繪圖例、新聞研究、設計影片，一個人努力開發了六個月。原本預計兩個月內完成，結果投入了整整一個冬天。
