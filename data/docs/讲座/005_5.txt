8331	{塔罗+命盤} {課程} 塔羅牌占卜	塔羅占卜課程 - 預知過去未來
香港美藉華人專業授教全套占卜課程，讓你成為下一代專業占卜師。
 { 引言 }
塔羅占卜可以預測"事業運" "感情運" "國運"，讓你趋吉避凶，掌握命運。
談事、創業、談朋友，出門前翻一翻塔羅牌，結果馬上出現。
在課程中，學生透過占卜案例和實戰去了解和應用這門進口的高等技術。
在學習塔羅的過程中，人類正在開發潛在的預知能力。
 { 塔羅兩大功能 }
一. 能知過去未來，為未來作有效準備！
二. 和心靈直接溝通(治療)了解、分析和解除心理矛盾阻障，重得快樂。
三. 占卜對事情分析(預測)預知未來、分析問題、建議選擇。
 { 你能獲得 }
浪仁老師給你的人生引路，親自傳授占卜絕學，學會趋吉避凶。
你將會有全面的預測能力去把握未來，預測"個人運勢"、"特別事情"、"社會動向"的發展。
你可以成為職業塔羅占卜師，收費作駐店及預約占卜服務。
你將會更了解身邊朋友的運勢，更為社會上知名人仕提供咨詢。
 { 課程內容 16小時制 }
I  ：功能。案例分享
II ：命卜種類。簡易占卜。謬論
III：歷史。顏色學。數字學。身心靈。四元素
IV：大牌講解
V ：小牌講解
VI  ：牌陣與問題
VII ：實戰
VIII：直覺練習
全程PPT課程，包括實例分析，真人講解示範。
同時，可選英語模式互動上課。
